OVERVIEW

밤 늪지 위를 뛰어다니며 날아다니는 파리를 잡는 60초 아케이드 게임입니다. 스페이스바를 꾹 눌러 점프 게이지를 채운 뒤 손을 놓으면 개구리가 뛰어오릅니다. 황금빛 큰 파리는 일반 파리보다 2배 점수를 줍니다. 게임 종료 후 점수는 로컬 랭킹과 온라인 리더보드에 동시에 기록됩니다.

제한 시간은 60초입니다. 남은 시간이 10초 미만이면 타이머가 빨간색으로 점멸합니다. 게임 종료 후 상위 5개 점수가 ranking.txt에 저장됩니다.

PYGAME 버전 호환성

이 게임은 pygame 버전에 따라 실행이 되지 않을 수 있습니다.
pygame.key.start_text_input() 및 pygame.TEXTINPUT 이벤트는 pygame 2.0 이상에서만 정상 지원됩니다. 구버전에서는 닉네임 입력 화면이 제대로 동작하지 않거나 오류가 발생할 수 있습니다.

pygame 버전상태비고
2.1.x 이상 (권장) ✔ 정상 실행 모든 기능 지원, IME 입력 안정적
2.0.x △ 대부분 동작 TEXTINPUT 지원, 일부 환경에서 IME 불안정
1.9.x 이하 ✘ 실행 불가 TEXTINPUT 이벤트 미지원 → 오류 발생

HOW TO RUN

# 1. 의존성 설치 (pygame 2.1 이상 권장)
pip install "pygame>=2.1"

# 설치된 버전 확인
python -c "import pygame; print(pygame.version.ver)"

# 2. 실행
python practice.py

leaderboard_client.py 모듈이 반드시 같은 디렉터리에 있어야 합니다. 오프라인 상태에서는 점수 업로드가 실패하더라도 로컬 랭킹은 정상 저장됩니다. PyInstaller로 빌드된 exe 환경도 지원합니다.
